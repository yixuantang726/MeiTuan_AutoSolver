# AutoSolver：快递派单优化的多策略擂台赛系统

AutoSolver 是一个融合多种组合优化算法与大语言模型决策机制的快递派单优化系统。针对"从80个骑手候选方案中选择最优派发集合"的NP难问题，系统创新性地采用**擂台赛竞争框架**，让贪心、启发式、整数规划等五种求解策略轮次竞争。

核心创新在于：**J-Agent 充当独立裁判而非求解策略参与者**。这样设计避免了 LLM 推理黑箱对优化结果的污染，确保每一步决策都有精确的数学根据。同时，通过 λ 参数的自适应调节，系统灵活平衡覆盖率与派单成本，实现显式的启发式规则体现。

系统支持冗余派发、跨轮资源规划、合单机会挖掘等功能，在300秒时间限制内通过多轮擂台赛找到高质量解。整个过程完全可追溯、可验证，适用于快递物流、任务调度等实际应用场景。

**关键词**：组合优化、多策略竞争、工具化LLM、可解释性
